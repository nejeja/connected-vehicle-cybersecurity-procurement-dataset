# Data

This folder contains the coding dataset used for the article:

**Connected Vehicles as Targets of AI-Enabled Hybrid Threats: Cybersecurity and Regulatory Implications for Public Institutions**

## Files

- `unified_coding_table.xlsx` – main Excel workbook containing the coded procurement records, codebook and summary sheet.
- `unified_coding_table.csv` – machine-readable export of the main coding table.
- `codebook.md` – explanation of coding categories and selected variables.

## Scope

The dataset contains public procurement records related to vehicles, public administration fleets and functions relevant to the Integrated Rescue System. The core Czech sample is based on NEN records from 2018–2026 and is supplemented by illustrative comparator examples from the United Kingdom and the Netherlands.

## Methodological limitation

The coding reflects only information visible in the reviewed public procurement documents. Absence of a term or requirement does not prove that the issue was not addressed elsewhere. It only means that the requirement was not visible in the reviewed public documentation.

Partially redacted, anonymised, unreadable or incomplete documents are treated as limitations of observability, not as proof of absence.
